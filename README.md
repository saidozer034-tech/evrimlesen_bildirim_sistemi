# Evrimleşen Bildirim Sistemi 🚀
Bu proje, üniversite final ödevi kapsamında Nesne Tabanlı Programlama (OOP) ve Tasarım Örüntüleri (Design Patterns) kullanılarak geliştirilmiştir.

## 🛠 Kullanılan Teknolojiler
- **Dil:** Java
- **CI/CD:** GitHub Actions
- **Mimari:** Design Patterns (Factory, Adapter, Decorator, Strategy, Observer)

## 📂 Proje Yapısı
- `src/`: Tüm Java kaynak kodları
- `docs/`: AI süreç raporları ve UML diyagramları
- `PATTERNS.md`: Uygulanan tasarım örüntülerinin detaylı açıklamaları
